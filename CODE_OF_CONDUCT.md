# Code of Conduct

This project adopts the **Contributor Covenant**, version 2.1, as its code of
conduct. The full text is published at:

- https://www.contributor-covenant.org/version/2/1/code_of_conduct/
- FAQ: https://www.contributor-covenant.org/faq

In short: we want participation in libVES.c to be a harassment-free, respectful
experience for everyone, regardless of background or identity. Be considerate and
professional in issues, pull requests, and all other project spaces.

## Reporting

To report unacceptable behavior, contact the maintainers at
**conduct@vesvault.com**. Reports will be reviewed and handled confidentially.

> Maintainers: confirm this inbox is monitored, or change it to a real address
> before publishing.

Enforcement follows the Contributor Covenant's guidelines, linked above.
